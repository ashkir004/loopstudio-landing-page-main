# Frontend Mentor - Loopstudios Landing Page

**Brief**

This repository contains a responsive landing page built to match the Loopstudios design challenge from Frontend Mentor. The project focuses on semantic HTML, modern CSS (mobile-first responsive layout), and a small JavaScript toggle for the mobile navigation.

**Features**

- Fully responsive layout: adapts from mobile to desktop.
- Mobile navigation with accessible toggle behavior.
- Clean, semantic markup and modular CSS using BEM-style class names.

**Tech Stack**

- HTML5
- CSS3 (Flexbox and CSS Grid)
- JS
